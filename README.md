# 极简记账本

一个纯前端的个人记账单页应用，双击 HTML 文件即可使用。

## 功能

- 记录收入 / 支出，支持餐饮、通勤、购物等分类，也可自定义
- 当月收入、支出、结余实时统计
- Chart.js 饼图展示各分类消费占比
- 数据存储于浏览器 LocalStorage，刷新不丢失
- 移动端自适应，界面简洁清爽

## 使用

直接在浏览器打开 `index.html`，无需安装任何依赖。

或者访问 [GitHub Pages 地址](https://xhswd.github.io/simple-accounting/)（部署后生效）。

## 技术栈

- HTML + Tailwind CSS（CDN）
- 原生 JavaScript
- Chart.js（CDN）
- LocalStorage 本地存储
